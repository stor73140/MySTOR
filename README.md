
STOCK RM APP

Application PWA pour gestion de stock remontées mécaniques.

Fonctions :
- Connexion utilisateur (Google Sheet)
- Stock mécanique
- Stock électrique
- Ajout / modification pièces
- Emplacement
- Historique via Google Sheet
- Admin avec onglet paramétrage

Installation :

1. Créer un repository GitHub
2. Importer tous les fichiers
3. Activer GitHub Pages
4. Ouvrir le lien sur smartphone
5. Ajouter à l'écran d'accueil

Compatible :
Android / iPhone
